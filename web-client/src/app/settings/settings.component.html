Delete all unassigned players

You can delete all players that are not assigned to a team.

This is useful when you want to do a cleaning of users. Bear in mind that, if you have enabled User Sync with the server, you may get again unassigned users if they still exist there.

This is a destructive action. Those players will also lose their statistics. Write below 'remove-all-unassigned' to enable the button.

Delete all statistics

You can delete all the statistics that are stored in this game: points and badges for all the players.

This is useful when you want to do a game reset and start over again. During the next synchronization with the server, you'll get only the history of resolved issues that is kept there.

This is a destructive action. All players will lose their statistics. Even after synchronization, the results may be different since the server might be storing data only for a limited period of time (e.g. the last month). Write below 'remove-all-stats' to enable the button.
